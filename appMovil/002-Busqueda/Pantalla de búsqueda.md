**Descripción del modulo**
Este modulo tiene como utilidad permitir la búsqueda de medicamentos por nombre o por marca, además de eso permite agregar filtros como [Existente, marca, venta libre, con receta].

**Especificaciones de Interfaz (UI/UX):**

- Ubicación: Este modulo estará ubicado debajo del header, y estará centrado. 
- Input de Texto: "Buscar medicamento o síntoma..."
- cono de Filtro: Para que el usuario pueda filtrar rápidamente entre "Venta libre" y "Con receta".
- Nota UX: Al tocar la barra, la pantalla podría transicionar a un modal de búsqueda que sugiera autocompletado para evitar errores ortográficos en nombres médicos.
