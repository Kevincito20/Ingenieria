#1Requisitos_funcionales 

## RFI-001: Localización y Telemetría de Máquinas

**Especificación:** El sistema debe mostrar un listado estructurado de máquinas expendedoras cercanas al usuario, ordenadas de forma ascendente por distancia ortodrómica, e integrar un indicador visual de telemetría que refleje el estado operativo en tiempo real (activa/inactiva).

## RFI-003: Acceso Directo a Última Compra

**Especificación:** El sistema debe consultar el historial transaccional para mostrar un acceso directo a la "Última compra", provisto de un disparador lógico (botón de repetir pedido) que valide condicionalmente la existencia de inventario en la máquina objetivo antes de habilitar la acción.

## RFI-004: Notificaciones del Ciclo de Vida de Recetas

**Especificación:** El sistema debe procesar y mostrar notificaciones personalizadas orientadas a eventos del ciclo de vida de la receta, incluyendo alertas de nueva prescripción disponible, advertencias cronológicas de vencimiento cercano y avisos de reabastecimiento de medicamentos previamente marcados como faltantes.

## RFI-005: Indicador Global de Recetas Vigentes

**Especificación:** El sistema debe iterar sobre el vector de prescripciones electrónicas del usuario y renderizar un indicador visual distintivo en la interfaz global si el algoritmo determina la existencia de recetas vigentes y habilitadas para uso inmediato.

## RFI-006: Identidad y Validación KYC

**Especificación:** El sistema debe inyectar el nombre de pila extraído del token de sesión en la cabecera de saludo y renderizar el avatar del usuario. Si el usuario carece de validación de identidad KYC completa, el sistema superpondrá un icono de alerta crítica e inhabilitará los flujos de dispensación restringida.

## RFI-007: Módulo de Máquinas y Recetas Frecuentes

**Especificación:** El sistema debe renderizar el módulo de "Tus máquinas frecuentes/recetas" implementando una cuadrícula responsiva de dos columnas, poblando dinámicamente cada tarjeta con el nombre genérico, la institución emisora y la fecha, con cifrado en tránsito para proteger la confidencialidad.

## RFI-008: Selector Global de Máquina Objetivo

**Especificación:** El sistema debe incorporar un modal o selector global persistente que fuerce al usuario a definir una máquina expendedora física objetivo, cuyo identificador único será anexado como parámetro obligatorio en todas las llamadas API subsiguientes de consulta de inventario.

## RFI-010: Tarjetas de Medicamento y Reglas de Negocio

**Especificación:** El sistema debe mapear los resultados del motor de búsqueda en tarjetas especializadas de medicamento, las cuales deberán exponer obligatoriamente el precio unitario, disponibilidad en la máquina seleccionada, gramaje y un indicador semántico sobre si el ítem requiere retención de receta electrónica.
