
El desarrollo del entorno web de este proyecto se centra en la gestión por parte del Doctor, farmacéutico, técnico o cualquier entidad previamente establecida, pueda administrar de manera sencilla e intuitiva.

- **1- Login:

	- Se propone una pantalla de inicio de sesión que permita validar de manera correcta la identidad del profesional encargado. 
	
- **2- Pantalla Principal: Lista de pacientes
	- Buscador por nombre/ cedula
	- Filtros (próximamente)
	
	- Lista / tabla de pacientes:
		- Nombre completo.
		- Edad.
		- Motivo de consulta.
		- Hora de cita.
		- Estado (filtro).
		- Botón "Ver".
		

- **3- Panel del paciente 
	- 3.1- **Información general del paciente.
	- **Resumen:
		- Motivo de consulta.
		- Ultima visita
		- Información sensible (Alergias, condiciones)
		
	- **3.1.2- Consultas previas:
		- Historial clínico del paciente (Información confidencial )
		- Lista de consultas anteriores
			- Fecha.
			- Diagnostico.
			- Tratamiento.
			- Doctor.

**ESTO VA DENTRO DEL PANEL DEL PACIENTE.

- **3- Nueva Consulta
	- Motivo de consulta
	- Síntomas.
	- Diagnóstico.
	- Tratamiento (Farmacéutico)
	- Notas adicionales.
		- Acciones:
			- Guardar
			- Guardar y generar receta.


- 5- **Exámenes:
	- Lista de Exámenes realizados:
		- Tipos (sangre, Rayos x)
		- Fecha
		- Resultados.
			- Botón: "Ver resultados"



- 6- **Recetas Electrónicas:
	- Medicamento
	- Dosis
	- Frecuencia
	- Duración.
	
	- Acciones: 
		- Generar receta
		- Enviar receta al paciente
