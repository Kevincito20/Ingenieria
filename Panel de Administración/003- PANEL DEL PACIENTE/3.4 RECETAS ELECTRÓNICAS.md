
La pestaña de **"Receta electrónica"** permite al doctor crear, gestionar y enviar recetas médicas digitales de manera rápida y segura, optimizando el proceso de prescripción.

Esta cuenta con dos módulos:

- _**Editor de receta:**_
    
    Espacio principal donde el doctor redacta la prescripción médica de forma clara y estructurada.
    
    Datos incluidos:
    
    - Medicamento
    - Dosis
    - Frecuencia
    - Duración del tratamiento
    - Indicaciones adicionales



- _**Acciones de envío:**_
    
    Permite gestionar la generación y envío de la receta al paciente.
    
    Funciones:
    
    - Generar receta electrónica
    - Enviar receta al paciente (digital)
    - Vista previa antes de enviar
    - Confirmación de envío




---

# **Requisitos Funcionales**



- **RF-001 Creación de Receta Electrónica:**
    - El sistema debe permitir al doctor crear una receta electrónica asociada a una consulta y paciente.


- **RF-002 Editor de Receta:**
    - El sistema debe proporcionar un editor estructurado que permita ingresar:
        - Medicamento.
        - Dosis.
        - Frecuencia.
        - Duración del tratamiento.
        - Indicaciones adicionales.



- **RF-003 Validación de Campos Obligatorios:**
    - El sistema debe validar que los campos esenciales (medicamento, dosis, frecuencia) estén completos antes de generar la receta.



- **RF-004 Vista Previa de Receta:**
    - El sistema debe permitir visualizar la receta antes de su envío para validación por parte del doctor.


- **RF-005 Envío de Receta al Paciente:**
    - El sistema debe permitir enviar la receta al paciente (APP)



- **RF-006 Confirmación de Envío:**
    - El sistema debe mostrar una confirmación clara cuando la receta haya sido enviada exitosamente.



- **RF-007 Edición de Receta Antes de Envío:**
    - El sistema debe permitir modificar la receta antes de ser enviada.



- **RF-008 Registro de Receta:**
    - El sistema debe almacenar la receta generada en el historial clínico del paciente.



---

# **Flujos de Excepciones**



- **FE-001 Campos Obligatorios Vacíos:**
    - Si el usuario intenta generar la receta sin completar campos obligatorios:
        - El sistema debe resaltar los campos faltantes.
        - Debe mostrar mensaje: _"Complete los campos obligatorios"_.
        - No debe permitir continuar.
