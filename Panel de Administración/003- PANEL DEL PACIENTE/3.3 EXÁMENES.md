
La pestaña de **"Exámenes"** permite al doctor visualizar y gestionar los estudios médicos realizados al paciente, facilitando el acceso rápido a resultados clínicos relevantes para la toma de decisiones.

Esta cuenta con dos módulos:

Esta cuenta con tres módulos:

- _**1 Listado de exámenes:**_  
    Visualización estructurada del historial de exámenes del paciente.


- _**1.1 Detalle del examen:**_  
    Muestra información completa del examen seleccionado.
    
    Datos incluidos:
    
    - Tipo de examen
    - Fecha
    - Resultado detallado
    - Observaciones



- _**Gestión de exámenes:**_  
    Permite acciones adicionales sobre los exámenes registrados.
    
    Funciones:
    
    - Adjuntar nuevos resultados (opcional)
    - Actualizar estado del examen
    - Eliminar o corregir registros (según permisos)

# **Requisitos Funcionales**



- **RF-001 Carga de Listado de Exámenes:**
    - El sistema debe recuperar y mostrar el historial de exámenes asociados al paciente seleccionado.



- **RF-002 Visualización de Listado de Exámenes:**
    - El sistema debe mostrar los exámenes en una lista estructurada incluyendo:
        - Tipo de examen.
        - Fecha.
        - Estado.



- **RF-003 Visualización de Detalle de Examen:**
    - El sistema debe permitir seleccionar un examen y visualizar su información completa:
        - Tipo de examen.
        - Fecha.
        - Resultado detallado.
        - Observaciones.



- ==**RF-004 Navegación entre Exámenes:**==
    - ==El sistema debe permitir cambiar entre exámenes sin recargar la página.==



- **RF-005 Adjuntar Resultados de Examen:**
    - El sistema debe permitir agregar nuevos resultados o archivos asociados a un examen existente.



- ==**RF-006 Actualización de Estado del Examen:**==
    - ==El sistema debe permitir actualizar el estado del examen (ej. pendiente, completado, revisado).==

# **Flujos de Excepciones**



- **FE-001 Error al Cargar Exámenes:**
    - Si ocurre un fallo al recuperar los exámenes:
        - El sistema debe mostrar mensaje: _"No se pudieron cargar los exámenes"_.
        - Debe permitir reintentar.
        - Debe evitar mostrar datos incompletos.



- **FE-002 Lista de Exámenes Vacía:**
    - Si el paciente no tiene exámenes registrados:
        - El sistema debe mostrar mensaje: _"No hay exámenes registrados"_.
        - Debe mostrar una vista vacía amigable.



- **FE-003 Error al Cargar Detalle del Examen:**
    - Si falla la carga del detalle:
        - El sistema debe mostrar mensaje: _"No se pudo cargar el detalle del examen"_.
        - Debe permitir reintentar sin salir del módulo.



- **FE-004 Error al Adjuntar Resultados:**
    - Si falla la carga de archivos o datos:
        - El sistema debe mostrar mensaje: _"No se pudo adjuntar el resultado"_.
        - Debe permitir reintentar.
        - No debe perder la información previamente ingresada.



- ~~**FE-005 Archivo Inválido o No Soportado:**~~
    - Si el archivo no cumple formato o tamaño:
        - El sistema debe mostrar mensaje: _"Archivo no válido"_.
        - Debe impedir la carga.
